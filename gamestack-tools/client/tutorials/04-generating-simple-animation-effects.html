Using the Effect Generator : A Two Step Process

Step 1:

Start the Squeeze-js Effects-Generator

  1. Download module
  2. NPM-Start

Step 2:

Load an existing .png and select and effect

  1. Load the .png
  2. Select your effect

Step 2:

Choose the start and end property-values

  1. Start Values
  2. End Values

Step 3:

Output the animation to .png format

  1. Output your animation as .png file
  2. It is now in the specified directory

*Now you can use this .png image as an animation in the SpriteMaker tool.
