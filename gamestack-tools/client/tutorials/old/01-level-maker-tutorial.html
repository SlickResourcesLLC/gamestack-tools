Creating a Gamestack: LevelMaker Scene

Step 1: Open the LevelMaker Tool

  1. Select New >> Level

Step 2: Select Quick-Load-Map-Images

  1. Load images of your choice, OR our image-resources from [DIR!]

Step 3: Select and redefine each level-object, now loaded by the Level-Maker.

  1. Select each item and edit, one-by-one

Step 4: Start Building your level.

  1. Notice the type property. *Be sure to select OR create a 'type' value for every Level-Sprite Object
  2. When your level materials are the right size
  3. Select, Point, and Click Sprites

Step 5: Output your level, with Save >> To File.

  1. Save your file, then navigate to the directory
  2. Right click and 'copy' the file to clipboard, then paste the file in a folder of your choice.
