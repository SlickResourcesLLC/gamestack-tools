Class: Sprite

Sprite

Takes an object of arguments and returns Sprite() object. Sprite() is a container for multiple Animations, Motions, and Sounds. Sprites have several behavioral functions for 2d-Game-Objects.

new Sprite(args)

Name Type Description

Properties
Name Type Description

src string

the source file for the GameImage:Sprite.image :: use a string / file-path

the size of the Sprite
position Vector

the position of the Sprite
padding Vector

the 'float-type' Vector of x and y padding to use when processing collision on the Sprite. A padding of new Vector(0.2, 0.2) will result in 1/5 of Sprite size for padding
selected_animation Animation

the selected_animation of the Sprite:: pass during creation or use Sprite.setAnimation after created
