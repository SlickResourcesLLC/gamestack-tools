Members

Gamestack

Object, instance of GamestackLibrary() : references Gamestack classes attaches to window object || module.exports (when loading via require)

JSManipulate

A collection of all the filters.

Methods

BlurFilter()

Blurs the image with Gaussian blur.

BrightnessFilter()

Adjusts the brightness of the image by going over to HSV values. Negative values decrease brightness while positive values increase brightness.

BumpFilter()

Embosses the edges of the image. This filter takes no parameters but can be applied several times for further effect.

CircleSmearFilter()

Smears out the image with circular shapes to create a painting style effect. The mix values sets the intensity of the effect. NOTE: This filter can be very slow, especially at higher densities/sizes. Use with caution.

ContrastFilter()

Adjusts the contrast of the image.

CrossSmearFilter()

Smears out the image with cross shapes to create a painting style effect. The mix values sets the intensity of the effect.

DiffusionFilter()

Diffuses the image creating a frosted glass effect.

DitherFilter()

Dithers the image to the specified number of colors. Setting color to false grayscales the image.

EdgeFilter()

Highlights the edges of the image.

EmbossFilter()

Embosses the image with a simulated light source. Angle and elevation sets the position of the light.

ExposureFilter()

Adjust simulated exposure values on the image.

FilterUtils()

Contains common filter functions.

GainFilter()

Adjusts the gain and bias of the image. Gain alters the contrast while bias biases colors towards lighter or darker.

GammaFilter()

Adjusts the gamma values of the image. Values over 1 increase the gamma while values over 0 decrease gamma.

GrayscaleFilter()

Sets the image to grayscale.

HueFilter()

Adjusts the hue of the image by going over to HSV values.

InvertFilter()

Inverts the colors of the image.

KaleidoscopeFilter()

Creates a kaleidoscope effect on the image. CenterX and CenterY specify the position in terms of ratios of width and height.

LensDistortionFilter()

Applies a fisheye lens distortion effect on the image. CenterX and CenterY specify the position in terms of ratios of width and height.

LineSmearFilter()

Smears out the image with line shapes to create a painting style effect. Mix specifies the intensity of the effect.

MaximumFilter()

Replaces every pixel with the maximum RGB value of the neighboring pixels. Each color is considered separately.

MedianFilter()

Replaces every pixel with the median RGB value of the neighboring pixels. Each color is considered separately.

MinimumFilter()

Replaces every pixel with the minimum RGB value of the neighboring pixels. Each color is considered separately.

NoiseFilter()

Creates random noise on the image, with or without color.

OilFilter()

Produces an oil painting effect on the image. NOTE: This filter can be very slow, especially at higher ranges. Use with caution.

OpacityFilter()

Changes the opacity of the image.

PinchFilter()

Pinches and whirls the image toward the center point. CenterX and CenterY specify the position in terms of ratios of width and height.

PixelationFilter()

Pixelates the image i.e. divides the image into blocks of color.

PosterizeFilter()

Posterizes the image, i.e. restricts the color values to a set amount of levels.

RGBAdjustFilter()

Adjust the factor of each RGB color value in the image.

SaturationFilter()

Adjusts the saturation value of the image. Values over 1 increase saturation while values below decrease saturation. For a true grayscale effect, use the grayscale filter instead.

SawtoothRippleFilter()

Creates ripples on the image horizontally/vertically in a sawtooth pattern.

SepiaFilter()

Creates a sepia effect on the image i.e. gives the image a yellow-brownish tone.

SharpenFilter()

Sharpens the image slightly. For increased effect, apply the filter multiple times.

SineRippleFilter()

Creates ripples on the image horizontally/vertically in a sine pattern.

SolarizeFilter()

Produces a solarization effect on the image.

SparkleFilter()

Generates a sparkle/sunburst effect on the image. CenterX and CenterY specify the position in terms of ratios of width and height.

SquareSmearFilter()

Smears out the image with square shapes to create a painting style effect. The mix values sets the intensity of the effect. NOTE: This filter can be very slow, especially at higher densities/sizes. Use with caution.

ThresholdFilter()

Divides the colors into black and white following the treshold value. Brightnesses above the threshold sets the color to white while values below the threshold sets the color to black.

TriangleRippleFilter()

Creates ripples on the image horizontally/vertically in a sine pattern.

TwirlFilter()

Twists the image around a given center point. CenterX and CenterY specify the position in terms of ratios of width and height.

VignetteFilter()

Creates a classical vignette effect on the image i.e. darkens the corners.

WaterRippleFilter()

Produces a water ripple/waves on the image. CenterX and CenterY specify the position in terms of ratios of width and height.
